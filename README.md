# archlinuxcn-dotfiles-collection

本仓库为收集热爱 Linux 桌面定制的用户 dotfiles 而生！

贡献要求：

- 至少一张具有代表性的桌面截图
- 提供 dotfiles 源仓库链接
- 没有任何桌面与窗口管理器限制
- 对发行版不做限制，虽然目前参与者大多是 archlinux 用户
